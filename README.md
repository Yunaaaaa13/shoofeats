# 🚀 Installation

Clone repository

```bash
git clone https://github.com/Yunaaaaaa13/shoofeats.git
```

Go into project

```bash
cd shoofeats
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open browser

```txt
http://localhost:3000
```

---

# 🔑 Environment Variables

Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```
